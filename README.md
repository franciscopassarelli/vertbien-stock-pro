Vertbien Stock Pro
Sistema de gestión de inventario diseñado para el control y seguimiento de productos.

🚀 Tecnologías
Frontend: React / Vite

Runtime: Bun

Lenguaje: TypeScript

Enrutamiento: TanStack Router

🛠️ Estructura del Proyecto
src/components: Interfaz de usuario, incluyendo formularios de login y modales para la gestión de productos y ventas.

src/routes: Configuración centralizada de las rutas de la aplicación.

src/hooks: Lógica de negocio para la manipulación de estados y datos del stock.

📝 Estado del Proyecto
Actualmente, el sistema cuenta con un flujo de autenticación simplificado para facilitar el acceso durante la etapa de desarrollo. La arquitectura está orientada a la escalabilidad de las funcionalidades de inventario.
